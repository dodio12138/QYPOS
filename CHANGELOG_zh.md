## [Unreleased]

### Added
- **PBKDF2 PIN 哈希**：使用 Node.js 内置 `crypto.pbkdf2Sync`（SHA-512，10万次迭代，随机盐值）对员工 PIN 进行哈希存储。旧明文 PIN 在首次登录时自动升级，对用户透明无中断。
- **PIN 隐私保护**：`GET /users` API 不再返回 `pin` 字段；后台账户管理页面不再显示 PIN 明文。
- **账户管理界面重构**：卡片式布局、角色彩色徽章（Owner🔴/Cashier🔵/Kitchen🟢）、行内编辑/重置PIN、行内删除确认、PIN 显示/隐藏切换、PIN 确认两次输入。
- **PIN 哈希单元测试**：9 个测试用例覆盖 hash/verify/upgrade/边界情况全流程。
- **共享测试辅助模块** (`tests/helpers.mjs`)：统一 API 请求封装、登录、环境管理、清理工具。
- **AGENTS.md**：项目级 AI 编码助手指南，包含技术栈、构建命令、架构决策、常见陷阱。
- **员工管理**：员工增删改查、排班、考勤记录、时薪管理。
- **排班系统**：周排班计划器，支持拖拽时间预设、实际出勤记录和工时转化率（营业额/小时）指标。
- **角色权限**：精细化收银员权限集；服务费调整/豁免、折扣、订单取消等操作均需显式授权。
- **POS 登录门禁**：前台所有操作（开台、加菜、打印、收款、清台）均需员工登录验证。
- **下单确认步骤**：堂食开台与外带创建前增加确认弹窗，防止误触。
- **Dojo Go 刷卡终端**：集成 Dojo Go 支付终端，支持 Pay-at-Counter 模式（Payment Intent + 终端会话生命周期）。
- **严格打印机路由**：厨房/收银指定打印机缺失、停用或无 IP 时直接报错，不再静默降级到其他设备。
- **仅新增菜品厨打锁定**：只锁定并打印新增菜品，已厨打的菜品不会重复出单。
- **热销菜下钻**：数据看板热销菜品支持多选合并趋势图，兼容历史数据中缺失 `item_id` 的情况。
- **报表增强**：新增星期几多选筛选；日期预设扩充为今日、昨天、近 7 天、近 30 天、本月、本周、上周、上月。
- **报表分析构成图**：数据分析新增售出单品、服务费、折扣、VAT 汇总，菜单分类点单次数/营业额构成图，联动高亮与可隐藏趋势虚线。
- **每日趋势悬浮卡片**：每日营业额/单量/客单价趋势图悬停信息改为日期 + 星期几卡片，并显示当前数据点相对前一个数据点的涨跌百分比。
- **每日趋势纵轴刻度**：每日趋势图新增纵轴刻度与横向网格线，金额与单量会自动使用紧凑格式显示。
- **指标卡片新版式**：后台看板和数据分析的数字卡片新增指标图标、右上角趋势百分比，以及主数字下方的实际变化量。
- **后台菜单体验优化**：菜单管理增加概览统计、搜索、上下架筛选、菜品上一页/下一页分页，并将备注词条管理升级为独立预设区域，支持按菜单分类设置适用范围。
- **完整多语言**：前台与后台界面完整覆盖中文 / 英文文案。
- **ESLint 配置**：根级扁平化 ESLint 配置，核心规则 `no-undef` 已接入 CI 流水线，可自动捕获 prop 传递遗漏类回归问题。
- **审计日志筛选**：后台审计日志支持用户、操作类型和精确时间范围组合筛选。
- **敏感设置二次验证**：修改税率等敏感设置需当前账号与 PIN 二次确认。
- **订单分页**：后台订单列表按每页 20 条分页浏览。

### Fixed
- **备注词条分类范围保存**：兼容历史菜单分类 UUID 格式，修复点击单个分类后刷新又回到“全部分类”的问题。
- **备注词条范围选择**：备注词条适用分类改为整颗按钮点击，修复点击单个分类应用时没有明显反应的问题。
- **夏令时日期偏移**：单品趋势图表日期轴现已与主报表营业额日期完全对齐，不再因夏令时提前或延后一天。
- **`locale` prop 崩溃**：修复了 `PosLogin`、`DiscountAdminModal`、`SettingsView` 等多个组件因 `locale` 属性未声明或未传递而导致的 `ReferenceError` 崩溃，已通过 ESLint 规则防护。
- **折扣封顶**：折扣金额现已封顶至订单小计，防止出现负数或异常总额。

### Changed
- **付款校验**：付款金额必须大于 0，找零不能为负，已关闭订单拒绝重复收款。
- **备份调度**：后台运维面板现已支持手动备份、自动备份计划、备份文件列表和下载。
